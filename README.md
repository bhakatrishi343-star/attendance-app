# Automated Attendance App

## Setup Instructions

Since Node.js/npm was not detected automatically, please follow these steps to run the project.

### Prerequisites
1.  Install **Node.js** (LTS) from [nodejs.org](https://nodejs.org/).
2.  Install **Expo CLI** globally (optional but recommended): `npm install -g expo-cli`.

### Backend Setup
1.  Navigate to `backend/`:
    ```bash
    cd backend
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the server:
    ```bash
    npm run dev
    # or
    node server.js
    ```

### Mobile App Setup
1.  Navigate to `mobile-app/`:
    ```bash
    cd mobile-app
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the app:
    ```bash
    npm start
    ```
